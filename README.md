# SentBe HRD Center

SentBe 사내 교육 플랫폼 — HRD 교육자료 허브

## 구조

```
sentbe-hrd-center/
├── index.html              # 메인 교육 플랫폼 페이지 (준비 중)
├── courses/                # HRD Agent로 제작된 교육 과정별 폴더
│   └── [주제명]/
│       ├── research.md     # 리서치 원본
│       ├── research.html   # 리서치 HTML 변환본
│       ├── curriculum.md   # 커리큘럼 설계
│       └── outputs/        # 추가 산출물 (PPT, 영상 스크립트 등)
└── references/             # 참고 자료 인덱스
    ├── claude-courses-index.md   # Claude Courses 코스 목록
    └── ai-clippings-index.md     # AI 클리핑 자료 목록
```
