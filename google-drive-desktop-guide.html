💡Google Drive for Desktop이란?

🖥️

탐색기에서 직접 접근

Finder(Mac) 또는 파일 탐색기(Windows)에서 Google Drive 파일을 로컬 폴더처럼 사용합니다.

⚡

실시간 자동 동기화

저장하는 순간 클라우드에 반영됩니다. 팀원들도 즉시 최신 파일을 확인할 수 있습니다.

🏢

HR Gdrive 공유 드라이브

팀 공유 드라이브(HR Gdrive)를 탐색기에서 바로 열고 저장할 수 있어 협업이 훨씬 편해집니다.

💾

스트리밍 방식으로 용량 절약

파일을 열 때만 다운로드하는 스트리밍 방식으로 PC 저장공간을 거의 사용하지 않습니다.

ℹ️
웹 브라우저와의 차이 — 기존 브라우저에서 drive.google.com 접속하던 방식 대신, 앱 설치 후에는 탐색기에서 G Drive 폴더가 직접 보입니다. Word, Keynote, Figma 등 어떤 앱에서도 Drive 파일을 열고 저장할 수 있습니다.

⚖️Mac vs Windows 주요 차이점

항목 🍎 Mac 🪟 Windows
설치 파일 GoogleDrive.dmg GoogleDriveSetup.exe
탐색기 통합 위치 Finder 왼쪽 사이드바 파일 탐색기 → "Google Drive (G:)"
상태 확인 메뉴바 (상단 오른쪽) 시스템 트레이 (오른쪽 하단)
오프라인 선택 파일 우클릭 → "오프라인에서 사용 가능" 파일 우클릭 → "오프라인에서 사용 가능"
검색 통합 Spotlight(⌘+Space)에서 Drive 파일 검색 Windows 검색에서 Drive 파일 검색
