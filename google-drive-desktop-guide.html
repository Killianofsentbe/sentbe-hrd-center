Drive for Desktop 완벽 가이드

Google Drive를 내 컴퓨터 폴더처럼 사용하세요.
설치부터 공유 드라이브 연동까지 한 번에 알아봅니다.

🖥️ Mac / Windows 지원 ☁️ 실시간 동기화 🤝 공유 드라이브 🔒 센트비 계정 연동

💡 Google Drive for Desktop이란?

🖥️

내 PC에서 직접 접근

파일 탐색기(Finder/Explorer)에서 Google Drive 파일을 로컬 폴더처럼 열고 저장할 수 있습니다.

⚡

실시간 동기화

저장과 동시에 클라우드에 자동 업로드됩니다. 다른 기기에서도 즉시 확인 가능합니다.

💾

스트리밍 vs 미러링

파일을 클라우드에서 스트리밍하거나 로컬에 다운로드해 오프라인 사용도 가능합니다.

🏢

공유 드라이브 연동

HR Gdrive 등 팀 공유 드라이브를 탐색기에서 바로 볼 수 있어 협업이 편리합니다.

ℹ️
웹 브라우저와의 차이점
기존에 browser.google.com/drive 로 접속하던 방식 대신, 앱을 설치하면 윈도우 탐색기 / Mac Finder에 G 드라이브가 나타납니다. Word, Excel, Figma 등 어떤 앱에서든 직접 열고 저장할 수 있습니다.

⬇️ 설치 방법

1

다운로드

Google 공식 사이트에서 운영체제에 맞는 버전을 다운로드합니다.

🔗 google.com/drive/download 접속 → "Drive for Desktop 다운로드" 클릭
2

설치 실행

Mac: 다운로드된 GoogleDrive.dmg를 열고 앱을 응용 프로그램 폴더로 드래그합니다.

Windows: GoogleDriveSetup.exe를 실행하고 지시에 따라 설치합니다.

3

센트비 계정으로 로그인

설치 후 앱이 실행되면 @sentbe.com 계정으로 로그인합니다. Google Workspace SSO가 적용되어 있어 비밀번호 없이 바로 인증됩니다.

💡 개인 Google 계정도 추가할 수 있습니다 (계정 전환 기능)
4

동기화 방식 선택

로그인 후 동기화 방식을 선택합니다. 대부분의 경우 스트리밍(권장)을 선택하면 됩니다.

📌 스트리밍: 클라우드 보관, 열 때만 다운로드 | 미러링: 전체 로컬 저장
5

탐색기에서 확인

Mac: Finder 왼쪽 사이드바에 Google Drive 항목이 생깁니다.
Windows: 파일 탐색기에 "G:" 또는 "Google Drive" 드라이브가 생깁니다.

⚙️ 스트리밍 vs 미러링 — 어떤 걸 써야 할까?

구분 ☁️ 스트리밍 (권장) 💾 미러링
파일 저장 위치 클라우드(Drive)에만 저장 클라우드 + 내 PC 로컬에 동시 저장
로컬 저장 공간 절약 거의 사용 안 함 소비 전체 파일 다운로드
오프라인 사용 제한적 개별 파일 선택 후 가능 가능 전체 파일 항상 접근
인터넷 필요 파일 열 때 필요 동기화 시에만 필요
추천 대상 대부분의 직원, 저장공간이 작은 노트북 대용량 파일을 자주 오프라인 사용하는 경우
⚠️
미러링 사용 시 주의
공유 드라이브(HR Gdrive 등)를 미러링하면 수십 GB의 파일이 모두 내 PC에 다운로드될 수 있습니다. 저장공간 여유가 충분한지 확인 후 선택하세요.

🏢 센트비 공유 드라이브 연동

✅
HR Gdrive 접근 권한
HR팀 직원은 HR Gdrive에 자동으로 접근 권한이 부여되어 있습니다. Google Drive for Desktop에 센트비 계정으로 로그인하면 탐색기에 공유 드라이브가 자동 표시됩니다.
1

공유 드라이브 표시 확인

Finder/탐색기에서 Google Drive → 공유 드라이브 폴더를 찾습니다. HR Gdrive, 전사 공유 드라이브 등 접근 권한이 있는 드라이브가 표시됩니다.

2

파일 열기 및 저장

공유 드라이브의 파일을 일반 폴더처럼 더블클릭해 열 수 있습니다. 저장하면 클라우드에 즉시 반영되어 팀원들과 실시간 공유됩니다.

💡 Google Docs/Sheets는 브라우저로 열립니다. Office 파일(.docx, .xlsx)은 로컬 앱으로 열립니다.
3

새 파일 업로드

로컬에서 만든 파일을 공유 드라이브 폴더에 드래그&드롭하거나 복사-붙여넣기 하면 자동으로 업로드됩니다.

⭐ 알아두면 유용한 기능

📌

파일 오프라인 사용

스트리밍 모드에서 특정 파일을 오프라인으로 사용하려면 파일 우클릭 → "오프라인에서 사용 가능"을 선택합니다.

🔗

공유 링크 복사

파일 우클릭 → "링크 복사"를 클릭하면 Google Drive 공유 링크가 클립보드에 복사됩니다. 탐색기에서 바로 공유 가능!

🔄

동기화 상태 확인

메뉴바(Mac) 또는 시스템 트레이(Windows)에서 Google Drive 아이콘을 클릭하면 현재 동기화 상태를 확인할 수 있습니다.

🔍

Spotlight / 검색 통합

Mac에서 Spotlight(⌘+Space), Windows에서 탐색기 검색을 통해 Drive 파일 내용까지 검색됩니다.

👤

다중 계정 지원

센트비 계정(@sentbe.com)과 개인 계정을 동시에 연결해 사용할 수 있습니다. 트레이 아이콘에서 계정 전환 가능.

📂

특정 폴더만 동기화

환경설정 → Google Drive → 동기화할 폴더를 개별 선택할 수 있어, 자주 쓰는 폴더만 빠르게 접근 가능합니다.

❓ 자주 묻는 질문

설치 후 Drive 아이콘이 탐색기에 안 보여요. ▼
앱이 실행 중인지 먼저 확인하세요. Mac은 메뉴바 상단에, Windows는 시스템 트레이에 Google Drive 아이콘이 있어야 합니다. 아이콘이 없으면 응용 프로그램에서 Google Drive를 다시 실행해 주세요. 재설치 후에도 안 보이면 재부팅을 시도해 보세요.
파일이 동기화되지 않고 계속 로딩돼요. ▼
인터넷 연결 상태를 먼저 확인하세요. 연결이 정상이라면 트레이 아이콘 클릭 → 설정 → "동기화 일시중지 후 재개"를 시도해 보세요. 그래도 해결되지 않으면 앱을 완전히 종료 후 재시작합니다.
파일을 삭제했는데 복구할 수 있나요? ▼
브라우저에서 drive.google.com 접속 → 왼쪽 메뉴 "휴지통"에서 30일 내 복구 가능합니다. 공유 드라이브 파일은 drive.google.com에서 해당 공유 드라이브 → 휴지통을 확인하세요.
개인 PC에도 설치할 수 있나요? ▼
네, 가능합니다. 센트비 계정(@sentbe.com)으로 로그인하면 회사 Drive와 공유 드라이브에 접근할 수 있습니다. 단, 보안 정책에 따라 회사 자료를 개인 PC에 다운로드할 때는 주의가 필요합니다.
Drive 저장 용량이 초과되면 어떻게 되나요? ▼
센트비는 Google Workspace Business 플랜을 사용하며, 각 계정에는 충분한 저장 용량이 제공됩니다. 공유 드라이브는 별도 용량 풀을 사용합니다. 개인 Drive 용량 부족 시 IT팀에 문의해 주세요.
Google Drive와 OneDrive를 동시에 사용해도 되나요? ▼
네, 동시 사용 가능합니다. 다만 같은 파일을 두 서비스에 모두 동기화하면 충돌이 발생할 수 있으므로 주의하세요. 센트비 공식 협업 도구는 Google Drive이므로 업무 파일은 Google Drive를 주로 사용하는 것을 권장합니다.
