아직 /clear만 쓰세요?

진짜 쓸모있는 슬래시 명령어 모음

🔄 세션 관리
/compact
대화 맥락을 압축해서 컨텍스트 절약
/clear는 대화를 통째로 날려요. /compact는 핵심 맥락만 남기고 압축해요.
/resume
터미널 껐다 켜도 이전 대화 이어가기
세션 복구 · 대화 이어하기
/context
현재 컨텍스트 사용량 확인
토큰 확인 · 용량 관리
💡 /clear 대신 /compact를 쓰세요 — 맥락을 유지하면서 공간을 확보할 수 있어요.
📋 작업 관리
/plan
복잡한 작업 전 실행 계획 수립
바로 '코드 짜줘'라고 하면 삽질할 수 있어요. /plan을 켜면 실행 없이 계획부터 세워요.
/diff
커밋 전 변경 내역 미리보기
커밋 전에 어떤 코드가 바뀌었는지 확인해요.
/cost
세션별 토큰 사용량/비용 확인
Input / Output / Total 비용을 실시간으로 확인해요.
💡 먼저 계획, 그 다음에 실행 — /plan으로 검토 후 승인하면 실수가 줄어요.
⚡ 성능 조절
/fast
빠른 응답 모드 토글
간단한 작업은 fast 모드로 빠르게 처리해요.
/effort
사고 깊이 조절 (low / medium / high)
복잡한 문제엔 high, 간단한 질문엔 low로 설정해요.
/model
모델 직접 선택 (Haiku / Sonnet / Opus)
작업 난이도에 맞게 모델을 골라 비용을 최적화해요.
💡 간단한 건 fast, 복잡한 건 high effort — 상황에 맞게 조절하면 비용과 속도 모두 잡을 수 있어요.
⚙️ 설정
/memory
CLAUDE.md 프로젝트 규칙 직접 편집
프로젝트, 스타일, 톤앤매너, 금지 사항 등을 저장해 매번 설명하지 않아도 돼요.
/init
프로젝트 파일 분석 후 CLAUDE.md 자동 생성
빈 문서 → CLAUDE.md를 자동으로 만들어줘요.
💡 매번 설명 안 해도 돼요 — /memory로 한번 설정해두면 Claude가 기억해요.
🧰 유틸
/copy
마지막 응답을 클립보드에 복사
드래그 없이 바로 복사해요.
/export
전체 대화를 파일로 내보내기
대화 전체를 저장하고 싶을 때 사용해요.
/doctor
설치 상태 자동 진단
뭔가 안 될 때 /doctor부터 실행하세요.
